HTML basic tags

Heading Tag

The <h> tags starts from h1 to h6 (<h1>, <h2>, <h3>, <h4>, <h5>, <h6>). Used for displaying headers in a webpage.

Example:

<h1>Hello World!</h1>

<h2>Hello World!</h2>

<h3>Hello World!</h3>

<h4>Hello World!</h4>

<h5>Hello World!</h5>

<h6>Hello World!</h6>

Output:

H1 - Hello World!

H2 - Hello World!

H3 - Hello World!

H4 - Hello World!

H5 - Hello World!
H6 - Hello World!

Paragraph Tag

The <p> tag is used to display a paragraph of text.

Example:

<p>Hello World!</p>

Output:

Hello World!

Break Tag

The <br /> tag is used to add a line break. It is an empty tag.

Example:

<p>Lorem ipsum dolor sit amet <br /> consectetur adipisicing elit. Blanditiis, minus.</p>

Output:

Lorem ipsum dolor sit amet

consectetur adipisicing elit. Blanditiis, minus.

Horzontal Line Tag

The <hr /> tag is used to add a horizontal line. It is an empty tag.

Example:

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis, minus.</p>
<hr />
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis, minus.</p>

Output:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis, minus.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis, minus.

Anchor Tag

The <a> tag can be used to reference another page either locally or on the internet. The reference need to be put inside the href (HyperText Reference) attribute of the <a> tag. In addition to the href attribute the <a> tag also has another attribute called target. By specifying target="_blank" we can make a link open in a new tab.

Example:

<a href="https://www.wikipedia.org/">Link to Wikipedia</a>
<a href="https://www.wikipedia.org/" target="_blank">Link to Wikipedia</a>

Output:

Link to Wikipedia
Link to Wikipedia

Image Tag

The <img /> tag is used for loading images into a webpage. The address (can be either local or on the internet) of the image needs to be put inside the src (source) attribute of the <img /> tag. The alt (alternate) attribute is just an alternate name for the image which is displayed when the image fails to load. It is an empty tag.

Example:

<img src="image.jpg" alt="landscape"/>

Output:

Successful

Failed to load

Form Tag

The <form> tag is used to a create HTML form. It is used along with other tags such as <input /> (empty tag) and <label>.
The <label> tag is used for labelling.
The <input /> tag can take input from the user. It has many attributes and one such attribute is type and can be used to set the type of the input, like, <input type="text"/> to take text input or <input type="radio"/> to take a radio input.

Example:

<form>
    <label>Name:</label>
    <input type="text"/>
    <br><br>
    <label>Age:</label>
    <input type="number"/>
    <br><br>
    <input type="submit"/>
</form>
